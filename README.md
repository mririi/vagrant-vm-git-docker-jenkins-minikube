# VM Initialization

VM initialization using vagrant.
## Contains
Git, Docker, Java 17, Jenkins, MiniKube

## Set Up

Install [Vagrant](hhttps://developer.hashicorp.com/vagrant/docs/installation) then run:

```bash
vagrant init
vagrant up --provision
```

## Usage

You can connect using ssh:
```bash
vagrant ssh
```
